CSS Dom Search

CSS Dom Search

This is a handy trick to quickly find the DOM nodes you need. In the Elements Panel, press Cmd + F (Ctrl + F on Windows) to open the search box.

Design Mode

Design Mode

This can help us test our design without editing the content manually in the source code.

Time to Paint

Time to Paint

This feature captures screenshots every few seconds to illustrate page load times.

Auditing Tools

Auditing Tools

  • Powered by Lighthouse
  • Click the "Audits" panel and select the + sign in the top left corner
  • Shows scores for the following categories:
    • Progressive Web App
    • Accessibility
    • Best Practices
    • Performance

Hide Elements Quickly

Hide Elements Quickly

Sometimes we need to hide elements in our design to check how it will look without them. Chrome DevTools give us the ability to inspect an element and type h from the keyboard to hide it by toggling CSS visibility property.
